# 1.會計科目餘額檔[ ACC_BALANCE ]

## 新增

| POST     | /ACC_BALANCE/Add | 新增 |       |                    |
| -------- | :----------- | :--: | :---: | ------------------ |
|          |              |      |       |                    |
| 欄位     | 型別         | 長度 | 必要  | 說明               |
| baseRequest     | object        |  |   | 基底請求資訊               |
| ├ id     | string | 36 | Y | 員工系統識別碼 |
| ├ employeeNo     | string | 50 | Y | 員工編號 |
| ├ name     | string | 100 | Y | 員工姓名 |
| ├ companyId     | string | 36 | Y | 公司系統識別碼 |
| ├ departmentId     | string | 36 | Y | 部門系統識別碼 |
| ├ email     | string | 255 | Y | 聯繫電子郵件 |
| ├ departmentIds[]     | list |  | C | 兼職部門系統識別碼集合 |
|   └ departmentId     | string        | 36 | C | 部門系統識別碼 |
|  └ roleNames[]    | list |  | Y | 角色名稱集合 |
|   └ name   | string        | 256 | Y | 角色名稱 |
| data   | object        |  | Y |  |
| ├ ACBL_COMPID | string       | 2  | Y | 公司代號 |
| ├ ACBL_YEAR | int       |   | Y | 會計年度           |
| ├ ACBL_MONTH | int       |   | Y | 會計月份 |
| ├ ACBL_CODE | string       | 12  | Y | 會計科目代號 |
| ├ ACBL_DBAL | decimal       |   |  | 期初借方餘額 |
| ├ ACBL_CBAL | decimal       |   |  | 期初貸方餘額 |
| ├ ACBL_DAMT1 | decimal       |   |  | 本月借方金額1 |
| ├ ACBL_DAMT2 | decimal       |   |  | 本月借方金額2 |
| ├ ACBL_CAMT1 | decimal       |   |  | 本月貸方金額1 |
| ├ ACBL_CAMT2 | decimal       |   |  | 本月貸方金額2 |
| ├ ACBL_DCNT | int       |   |  | 本月借方筆數 |
| └ ACBL_CCNT | int       |   |  | 本月貸方筆數 |

## 查詢

| POST     | /ACC_BALANCE/Query | 查詢 ||                             |
| -------- | ------------ | ---- | ----- | ------------------ |
|          |              |      |       |                    |
| 欄位     | 型別         | 長度 | 必要  | 說明               |
| 略... |  |  | Y |  |
| data   | object |  | Y |  |
| ├ ACBL_COMPID | string       | 2  | Y | 公司代號 |
| ├ ACBL_YEAR | int       |   | Y | 會計年度           |
| ├ ACBL_MONTH | int       |   |  | 會計月份 |
| └ ACBL_CODE | List<string>       |    |  | 會計科目代號 |

```JSON
會計月份如果填入 0代表查詢全年度，會計科目代號未填寫則會全選。
查詢該年度全年
{ 
  略 ...,
  "data": 
	{ 
	"ACBL_COMPID":"01",
	"ACBL_YEAR": "112",
	"ACBL_MONTH":"0",
	"ACBL_CODE":[]
	}
}
查詢指定年月
{ 
  略 ...,
  "data": 
	{ 
	"ACBL_COMPID":"01",
	"ACBL_YEAR": "112",
	"ACBL_MONTH":"8",
	"ACBL_CODE":[]
	}
}
查詢指定年月且指定會計科目代號
{ 
  略 ...,
  "data": 
	{ 
	"ACBL_COMPID":"01",
	"ACBL_YEAR": "112",
	"ACBL_MONTH":"8",
	"ACBL_CODE":["012345678901", ...]
	}
}
查詢該年度全年且指定會計科目代號
{ 
  略 ...,
  "data": 
	{ 
	"ACBL_COMPID":"01",

	"ACBL_MONTH":"0",
	"ACBL_CODE":["012345678901", ...]
	}
}
```
