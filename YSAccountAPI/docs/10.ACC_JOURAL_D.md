# 10.常用分錄明細檔[ ACC_JOURAL_D ]

## 新增

| POST     | /ACC_JOURAL_D/Add | 新增 |       |                    |
| -------- | :----------- | :--: | :---: | ------------------ |
|          |              |      |       |                    |
| 欄位     | 型別         | 長度 | 必要  | 說明               |
| baseRequest     | object        |  |   | 基底請求資訊               |
| ├ id     | string | 36 | Y | 員工系統識別碼 |
| ├ employeeNo     | string | 50 | Y | 員工編號 |
| ├ name     | string | 100 | Y | 員工姓名 |
| ├ companyId     | string | 36 | Y | 公司系統識別碼 |
| ├ departmentId     | string | 36 | Y | 部門系統識別碼 |
| ├ email     | string | 255 | Y | 聯繫電子郵件 |
| ├ departmentIds[]     | list |  | C | 兼職部門系統識別碼集合 |
|   └ departmentId     | string        | 36 | C | 部門系統識別碼 |
|  └ roleNames[]    | list |  | Y | 角色名稱集合 |
|   └ name   | string        | 256 | Y | 角色名稱 |
| data   | object        |  | Y |  |
| ├ JOUD_COMPID | string       | 36  | Y | 公司代號 |
| ├ JOUD_CODE | string       | 4  | Y | 常用分錄代號           |
| ├ JOUD_SEQ | int       |   | Y | 明細序號 |
| ├ JOUD_ACCD | string       | 12  |  | 會計科目代號 |
| ├ JOUD_MEMO | string       | 40  |  | 摘要 |
| ├ JOUD_DEPTID | string       | 36  |  | 部門代號 |
| ├ JOUD_TRANID | string       | 10  |  | 對象代號1 |
| ├ JOUD_INVNO | string       | 20  |  | 憑證號碼2 |
| ├ JOUD_DAMT | decimal       |   |  | 借方金額1 |
| └ JOUD_CAMT | decimal       |   |  | 貸方金額2 |


```JSON
{ 
"baseRequest": 
	{ 
	"id": "a4e57477-10f9-4698-989f-897728857d25", 
	"employeeNo": "11001",

	"departmentId": "5094cd3b-ca22-41ca-9d42-68f4aeac408c", 
	"email": "jack@yunsancorp.com", 
	"departmentIds": [
		{ 
		"departmentId": "5990dd48-fcfc-468a-a763-4d33602bf53f"   } 
	], 
	"roleNames": [ 
		{ "name": "Sales" }, 
		{ "name": "Admin" } 
	] 
}, 

"data":  
	{ 
	"JOUD_COMPID": "01",
	"JOUD_CODE":"A00",
	"JOUD_SEQ":1,
	"JOUD_ACCD":"ACCD",
	"JOUD_MEMO":"MEMO",
	"JOUD_DEPTID":"DEPTID",
	"JOUD_TRANID":"TRANID",
	"JOUD_INVNO":"INVNO",
	"JOUD_DAMT":"220.0",

	}
}
```
