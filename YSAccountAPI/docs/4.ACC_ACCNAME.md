# 4.會計科目主檔[ ACC_ACCNAME ]

## 新增

| POST     | /ACC_ACCNAME/Add | 新增 |       |                    |
| -------- | :----------- | :--: | :---: | ------------------ |
|          |              |      |       |                    |
| 欄位     | 型別         | 長度 | 必要  | 說明               |
| baseRequest     | object        |  |   | 基底請求資訊               |

| ├ ACNM_E_NAME | string       | 40  |  | 英文名稱 |
| ├ ACNM_KIND | string       | 1  |  | 會計大分類代號 |
| ├ ACNM_CODE | string       | 1  |  | 會計中分類代號 |

| POST     | /ACC_ACCNAME/QueryByRange | 查詢 ||                             |
| -------- | ------------ | ---- | ----- | ------------------ |

| 欄位     | 型別         | 長度 | 必要  | 說明               |
| 略... |  |  | Y |  |
| data   | object |  | Y |  |
| ├ ACNM_QUERY_TYPE | int       |   |  | 查詢範圍 0 or 1.主科目, 2.包含子科目-二階, 3.包含子科目-三階 |
