# 12.部門年度預算檔[ ACC_BUDGET ] 目前 Table 不存在

## 新增

| POST     | /ACC_BUDGET/Add | 新增 |       |                    |
| -------- | :----------- | :--: | :---: | ------------------ |
|          |              |      |       |                    |
| 欄位     | 型別         | 長度 | 必要  | 說明               |
| baseRequest     | object        |  |   | 基底請求資訊               |
| ├ id     | string | 36 | Y | 員工系統識別碼 |
| ├ employeeNo     | string | 50 | Y | 員工編號 |
| ├ name     | string | 100 | Y | 員工姓名 |
| ├ companyId     | string | 36 | Y | 公司系統識別碼 |
| ├ departmentId     | string | 36 | Y | 部門系統識別碼 |
| ├ email     | string | 255 | Y | 聯繫電子郵件 |
| ├ departmentIds[]     | list |  | C | 兼職部門系統識別碼集合 |
|   └ departmentId     | string        | 36 | C | 部門系統識別碼 |
|  └ roleNames[]    | list |  | Y | 角色名稱集合 |
|   └ name   | string        | 256 | Y | 角色名稱 |
| data   | object        |  | Y |  |
| ├ BUDG_COMPID | string       | 36  | Y | 公司代號 |
| ├ BUDG_YEAR | int       |   | Y | 年度           |
| ├ BUDG_MONTH | int       |   | Y | 月份 |
| ├ BUDG_DEPTID | string       | 36  | Y | 部門代號 |
| ├ BUDG_ACNMID | string       | 12  | Y | 會計科目代號 |
| ├ BUDG_B_AMT | decimal       |   |  | 預算金額 |
| ├ BUDG_R_AMT | decimal       |   |  | 實際金額 |
| ├ BUDG_D_AMT1 | decimal       |  |  | 分攤金額 |
| └ BUDG_D_AMT2 | decimal       |  |  | 分攤金額1 |
| (不用輸) BUDG_A_USER_ID | string       | 36  |  | 登錄者代號 |
| (不用輸) BUDG_A_USER_NM | string       | 20  |  | 登錄者姓名 |
| (不用輸) BUDG_A_DATE | datetime       |   |  | 登錄日期 |
