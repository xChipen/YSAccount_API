# 15.應收票據檔[ ACC_RE_CHECK ]

## 新增

| POST     | /ACC_RE_CHECK/Add | 新增 |       |                    |
| -------- | :----------- | :--: | :---: | ------------------ |
|          |              |      |       |                    |
| 欄位     | 型別         | 長度 | 必要  | 說明               |
| baseRequest     | object        |  |   | 基底請求資訊               |
| ├ id     | string | 36 | Y | 員工系統識別碼 |
| ├ employeeNo     | string | 50 | Y | 員工編號 |
| ├ name     | string | 100 | Y | 員工姓名 |
| ├ companyId     | string | 36 | Y | 公司系統識別碼 |
| ├ departmentId     | string | 36 | Y | 部門系統識別碼 |
| ├ email     | string | 255 | Y | 聯繫電子郵件 |
| ├ departmentIds[]     | list |  | C | 兼職部門系統識別碼集合 |
|   └ departmentId     | string        | 36 | C | 部門系統識別碼 |
|  └ roleNames[]    | list |  | Y | 角色名稱集合 |
|   └ name   | string        | 256 | Y | 角色名稱 |
| data   | object        |  | Y |  |
| ├ RECK_COMPID | string       | 36  | Y | 公司代號 |
| ├ RECK_NO | string       | 14  | Y | 票據號碼 |
| ├ RECK_REC_DATE | datetime       |   |  | 收票日           |
| ├ RECK_DUE_DATE1 | datetime       |   |  | 到期日           |
| ├ RECK_DEPTID | string       | 8  |  | 不知道文件上沒有但是資料庫有           |
| ├ RECK_CUSTID | string       | 10  |  | 客戶代號           |
| ├ RECK_AMT | decimal       |   |  | 票據金額           |
| ├ RECK_DUE_BANK | string       | 10  |  | 付款行           |
| ├ RECK_ACNO | string       | 16  |  | 付款行帳號           |
| ├ RECK_SAV_FLG | string       | 1  |  | 託收區分           |
| ├ RECK_SAV_DATE | datetime       |   |  | 託收日期           |
| ├ RECK_SAV_BANK | string       | 12  |  | 託收行會計科目代號           |
| ├ RECK_AREA_FLG | string       | 1  |  | 地區別           |
| ├ RECK_DUE_DATE2 | datetime       |   |  | 預定抵用日           |
| ├ RECK_DUE_FLG | string       | 1  |  | 兌現區分           |
| ├ RECK_DUE_DATE3 | datetime       |   |  | 實際兌現日           |
| ├ RECK_VOU_DATE | datetime       |   |  | 兌現開立傳票日期           |
| ├ RECK_VOUNO | string       | 13  |  | 兌現傳票號碼           |
| ├ RECK_C_VOUNO | string       | 13  |  | 立帳傳票號碼           |
| └ RECK_C_SEQ | int       |   |  | 立帳傳票明細序號           |
